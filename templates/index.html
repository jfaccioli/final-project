SEO Keyword Clustering application built with Flask API


This application aims to experiment machine learning to classify website queries in automatically generated clusters. Cluster names are also automatically generated so we can explore our query performances more easily. When building new pages or optimising our SEO (Search Engine Optimisation), we can have a better idea on what keyword families will perform better.
